# 🛒 ShopNow — Full Stack MERN eCommerce

A production-ready eCommerce application built with the MERN stack.

## 🚀 Tech Stack

- **Frontend:** React.js, Redux Toolkit, Tailwind CSS (Coming Soon)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Authentication:** JWT (Access + Refresh Tokens)
- **Image Storage:** Cloudinary
- **Payment:** Razorpay (Coming Soon)

## ✨ Features

### Completed

- ✅ User Authentication (Register/Login/Logout)
- ✅ JWT Access Token + Refresh Token
- ✅ HTTP-Only Cookie Security
- ✅ Role-Based Authorization (User/Admin)
- ✅ Product CRUD (Admin)
- ✅ Image Upload to Cloudinary
- ✅ Product Search, Filter, Sort, Pagination
- ✅ Cart System (Add/Update/Remove/Clear)
- ✅ Order System (Place Order/Track Status)
- ✅ Order Status Management (Admin)
- ✅ Stock Management (Auto decrease/restore)

### Coming Soon

- 🔜 Product Reviews & Ratings
- 🔜 Admin Dashboard
- 🔜 React Frontend
